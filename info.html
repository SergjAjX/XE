📖О проекте
▼

«Диабет-Помощник» — веб-приложение для людей с сахарным диабетом 1 типа и их родителей. Помогает точно рассчитать дозу инсулина на приём пищи с учётом углеводов (ХЕ), текущего сахара и активного инсулина в крови (IOB).

Приложение работает прямо в браузере — без установки, без регистрации, без передачи данных на сервер. Все данные хранятся на вашем устройстве. Текущая версия: v7.3 Pro

⭐Ключевые возможности
▼
  • 💉
    Расчёт болюса — учитывает ХЕ, текущий сахар, порог коррекции и активный инсулин IOB. Использует «Правило 100».
  • 💊
    Активный инсулин (IOB) — автоматически сканирует историю уколов за последние N часов и вычитает остаток из дозы. Исключает стекинг инсулина.
  • 🔍
    Поиск продуктов — фильтрация списка по вводу + автозаполнение веса из названия («Гречка 200» → 200 г).

«🧮 Калькулятор» → шестерёнка ⚙️ вверху → введите целевой диапазон сахара, Коэффициент на ХЕ, Фактор чувствительности и Длительность инсулина.

⚙️Подробная настройка коэффициентов
▼

🎯 Целевой диапазон сахара

Значения, которые считаются нормой лично для вас. Приложение использует их для цветовой индикации и расчёта дозы коррекции.

  • Нижняя граница (по умолчанию 5.0 ммоль/л) — ниже считается гипогликемия.
  • Верхняя граница (по умолчанию 7.0 ммоль/л) — выше начинается коррекция.
  • Типичные рекомендации ADA для взрослых: 3.9–10.0 ммоль/л. Для беременных и детей — только с врачом.

🍞 Коэффициент на ХЕ (углеводный коэффициент, УК)

Сколько единиц инсулина нужно на 1 ХЕ (= 12 г углеводов).

  • Пример: УК = 1.5, вы съели 3 ХЕ → болюс на еду = 1.5 × 3 = 4.5 ед.
  • Обычный диапазон: 0.5–2.5 ед/ХЕ. Утром обычно выше (феномен рассвета).
  • По умолчанию в приложении: 1.0 — замените на свой!

💧 Фактор чувствительности к инсулину (ФЧИ / ISF)

На сколько ммоль/л одна единица инсулина снижает сахар.

  • Расчёт по «Правилу 100»: ФЧИ = 100 ÷ суточная доза инсулина (ед/сут).
  • Пример: суточная доза 40 ед → ФЧИ = 100 ÷ 40 = 2.5 ммоль/л на 1 ед.
  • Обычный диапазон: 1.5–4.0 ммоль/л. По умолчанию: 2.0 — замените!

⏳ Длительность действия инсулина

Через сколько часов инсулин полностью перестаёт работать. Используется для расчёта IOB.

  • Ультракороткие аналоги (НовоРапид, Хумалог, Апидра): 3–4 часа.
  • Короткий инсулин (Актрапид, Хумулин R): 5–7 часов.
  • По умолчанию в приложении: 4 часа. Уточните у врача какой инсулин вы используете.

📏 Порог коррекции

Уровень сахара, выше которого приложение добавляет дозу коррекции к болюсу на еду.

  • Обычно совпадает с верхней границей целевого диапазона (например, 7.0 ммоль/л).
  • Если сахар ниже порога — коррекция не добавляется.
  • Для коррекции высокого сахара без еды — выберите «Коррекция» в корзине продуктов.

🔔 Уведомления браузера

  • При первом расчёте браузер запросит разрешение — нажмите «Разрешить».
  • Если пропустили: адресная строка → 🔒 замок → «Разрешения» → «Уведомления: Разрешить».
  • iPhone/iPad (Safari): добавьте страницу на домашний экран (Share → «На экран Домой») — только тогда работают push-уведомления.
  • Убедитесь что в настройках телефона для браузера уведомления включены.
💊Активный инсулин (IOB) — подробно
▼
💡
Что такое IOB?Insulin On Board — количество инсулина, которое ещё остаётся активным в крови после предыдущего укола. Если не учитывать IOB — можно ввести лишний инсулин и получить гипогликемию. Это называется «стекинг инсулина» (insulin stacking) — одна из самых частых причин опасного падения сахара.

📐 Как приложение считает IOB

Формула линейного распада:

Формула
IOB = Доза × (1 − Время_прошло / Длительность_инсулина)
Пример: укол 4 ед, прошло 2 часа, длительность 4 часа
IOB = 4 × (1 − 2/4) = 4 × 0.5 = 2 ед остаётся активным

Приложение сканирует все уколы за последние N часов (N = длительность инсулина) и суммирует остатки по каждому уколу.

🔢 Пример полного расчёта с IOB

Ситуация
Сахар: 9.5 ммоль/л | Цель: 6.0 | Еда: 3 ХЕ
Коэффициент на ХЕ: 1.5 | ФЧИ: 2.5 | IOB: 1.2 ед
Болюс на еду: 3 × 1.5 = 4.5 ед
Коррекция: (9.5 − 6.0) / 2.5 = 1.4 ед
Итого без IOB: 4.5 + 1.4 = 5.9 ед
Минус IOB: 5.9 − 1.2 = 4.7 ед — итоговая доза
✅ Без учёта IOB вы бы ввели на 1.2 ед больше → риск гипо!

❓ Если вы не уверены — как ввести IOB

  • Автоматически: нажмите «Авто» рядом с полем IOB — приложение само просканирует историю уколов. Проверьте результат во вкладке Debug.
  • Вручную: введите цифру сами. Если знаете что 2 часа назад вводили 3 ед → при ФЧИ 4ч остаток ≈ 1.5 ед.
  • Если не знаете / не уверены: введите 0. Лучше немного перестраховаться и проверить сахар через час, чем угадывать.

⚠️ Когда IOB особенно важен

  • Частые перекусы (менее 3–4 часов между едой).
  • Коррекция высокого сахара — обязательно проверьте IOB перед уколом.
  • После физической нагрузки — чувствительность к инсулину выше, IOB опаснее.
  • Ночные расчёты — когда сложнее контролировать гипо.
⚠️
IOB — только болюсный инсулинПриложение учитывает только болюсные (короткие) уколы из истории. Базальный инсулин (длинный) в расчёт не входит — его вводите по отдельной схеме врача.
🚫
Не делайте коррекцию чаще чем раз в 3–4 часаДаже если сахар кажется высоким — дождитесь пока предыдущий укол отработает. Два укола подряд с коротким интервалом — главная причина тяжёлой гипогликемии.
✅
Правило безопасной коррекцииСделали укол → подождали 3–4 часа → сахар всё ещё высокий → тогда корректируйте снова. Именно для этого в приложении есть таймер.

❓Частые вопросы (FAQ)
▼
Где взять мои коэффициенты (на ХЕ, чувствительности, длительность)? ▼
Только от вашего лечащего врача-эндокринолога. Это строго индивидуальные параметры. Длительность инсулина зависит от препарата — уточните какой именно вы используете. В приложении установлены безопасные значения по умолчанию (УК 1.0, ФЧИ 2.0, длительность 4 ч) — их обязательно замените на свои.
Почему приложение вычло из дозы меньше/больше чем я ожидал? ▼
Откройте вкладку Debug — там показан детальный расчёт IOB: какие уколы учтены, сколько прошло времени, сколько осталось. Проверьте что все болюсные уколы внесены в историю и что длительность инсулина в настройках соответствует вашему препарату.
Почему таймер не звонит? ▼
Проверьте: 1) таймер запустился (отсчёт идёт); 2) браузер разрешил уведомления для сайта (замок в адресной строке → Разрешения); 3) звук устройства включён; 4) на iPhone — добавьте страницу на домашний экран для работы push-уведомлений.
Как сделать резервную копию и перенести данные? ▼
Кнопка «💾 Экспорт» в шапке → скачается JSON-файл со всей историей. На новом устройстве нажмите «Импорт» и выберите тот же файл. Делайте резервную копию минимум раз в неделю — данные в браузере могут удалиться при очистке кэша.
Как откорректировать высокий сахар без еды? ▼
В корзине продуктов выберите пункт «Коррекция». Приложение рассчитает только дозу коррекции с учётом IOB, без болюса на еду. Обязательно проверьте IOB перед коррекцией — возможно предыдущий инсулин ещё работает.
Работает ли приложение на iPhone / Android? ▼
Да, адаптировано для мобильных. На iPhone: Safari → «Поделиться» → «На экран Домой» — работает как нативное приложение с уведомлениями. На Android добавьте в Chrome через меню → «Добавить на главный экран».
Учитывается ли базальный инсулин? ▼
Нет. IOB считается только для болюсного (короткого) инсулина. Базальный (длинный) вводится по отдельной схеме и в расчёт не входит. Не вносите дозу базального в историю уколов — это исказит расчёт IOB.
TIR 63% — это хорошо или плохо? ▼
По международным стандартам (ADA/EASD): TIR >70% — хороший результат для взрослых с СД1. Для пожилых и детей цели могут отличаться. Ваш показатель обсудите с эндокринологом — он учтёт вашу конкретную ситуацию.

Не заменяет врачаПриложение — вспомогательный инструмент. Все решения по изменению доз инсулина, коэффициентов и целевых уровней принимаются только после консультации с эндокринологом.
⚡
Гипогликемия — действуйте немедленноПри сахаре <4.0 ммоль/л: немедленно съешьте 15 г быстрых углеводов (4 кусочка сахара / ½ стакана сока / 5 таблеток глюкозы). Не откладывайте — читайте советы приложения потом. Через 15 минут повторите замер.
🔬
Высокий сахар (>15 ммоль/л) — проверьте кетоныПеред коррекционным уколом при сахаре выше 15 ммоль/л — проверьте уровень кетонов в крови/моче. Вводить большую дозу инсулина при кетоацидозе без контроля опасно.
⏳
Не колите повторно раньше чем через 3–4 часаСтекинг инсулина (несколько уколов подряд с коротким интервалом) — главная причина тяжёлой гипогликемии. Используйте IOB и таймер чтобы контролировать интервал.
🏃
После физической нагрузкиЧувствительность к инсулину повышается на 2–4 часа после тренировки. Рассчитанная доза может оказаться слишком большой. Снижайте дозу и чаще измеряйте сахар — конкретные цифры согласуйте с врачом.
🔒
Данные хранятся только на вашем устройствеПриложение не отправляет данные врачу автоматически. Используйте экспорт JSON и кнопку «Поделиться» чтобы передать историю специалисту.
👶
Для детей — только под контролем взрослыхКоэффициенты у детей меняются быстро (рост, болезни, физическая активность). Настройки должны регулярно пересматриваться с педиатром-эндокринологом.
