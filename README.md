📖 О проекте

«Диабет-Калькулятор» — это удобное веб-приложение для людей с сахарным диабетом 1 типа и их родителей. Оно помогает точно рассчитать дозу инсулина на прием пищи (болюс) с учетом углеводов (ХЕ), текущего уровня сахара и индивидуальных настроек (коэффициент на ХЕ, фактор чувствительности). Приложение автоматически запускает таймер для постпрандиального контроля и дает умные рекомендации.

⭐ Ключевые возможности

· ✅ Расчет болюса — учитывает углеводы в еде (стандартные и свои продукты), текущий сахар и активный инсулин, используя «Правило 100».
· ✅ Умный таймер 2 часа — автоматически запускается после расчета, напоминает сделать контрольный замер (звук, вибрация, push-уведомления). Сохраняется после закрытия вкладки.
· ✅ Постпрандиальный анализ — запись сахара через 1/2/3 часа после еды с цветовой индикацией (✅ Отлично / 🟡 Коррекция / 🔴 Опасно) и конкретными советами по действиям.
· ✅ Графики и статистика — визуализация уровня сахара, углеводов (г), процент попадания в целевой диапазон и распределение замеров по часам.
· ✅ История и автосохранение — все данные (продукты, замеры, дозы, время) сохраняются в браузере и не теряются при обновлении страницы.

🚀 Быстрый старт (За 5 минут)

1. Настройте коэффициенты: Перейдите на вкладку «🧮 Калькулятор» → нажмите на шестеренку настроек вверху. Введите свои целевые значения (по умолчанию 5.0-7.0), Коэффициент на ХЕ (ед/ХЕ) и Фактор чувствительности (ммоль/л). Все цифры должны быть согласованы с врачом!
2. Добавьте продукты: Выберите блюдо из списка, укажите вес в граммах и нажмите «➕ Добавить». Можно собрать полноценный прием пищи из нескольких позиций.
3. Рассчитайте дозу: Введите текущий уровень сахара (глюкометр или НМГ) и нажмите «💉 Рассчитать дозу». Приложение покажет: ХЕ, дозу на еду, дозу на коррекцию и итоговую дозу инсулина.
4. Запустите таймер: После расчета автоматически появится таймер на 2 часа. Нажмите «▶️ Старт».
5. Запишите результат: Через 2 часа (придет уведомление) измерьте сахар, перейдите на вкладку «🍽️ После еды», выберите «2 часа», введите значение и нажмите «💾 Сохранить». Получите оценку и рекомендацию.

📱 Руководство пользователя (Подробно)

1️⃣ Вкладка «🧮 Калькулятор»

· Продукты: стандартный список + кнопка «+» для добавления своих продуктов (нужно указать название и количество углеводов на 100 г).
· История: внизу вкладки отображаются все ваши расчеты и замеры с временными метками и цветовым статусом. Кнопка «Очистить» удаляет все данные безвозвратно.
· Расчет: учитывает только болюс на еду и коррекцию при сахаре выше Порога коррекции (в настройках). Активный инсулин на данный момент не вычитается (требуется ручной ввод в одном из следующих обновлений).

2️⃣ Вкладка «🍽️ После еды»

· Советы: При низком сахаре (<4.0) приложение посчитает, сколько граммов быстрых углеводов нужно съесть (например, 15 г = ½ стакана сока). При высоком (>10.0) — рассчитает примерную дозу для коррекции (на основе вашего фактора чувствительности).
· Статистика: внизу вкладки отображается процент ваших замеров, попавших в целевой диапазон (настраивается).

3️⃣ Вкладка «📊 Графики»

· Основной график (синие столбцы — углеводы, красная линия — сахар). Фильтры по диапазону: Сегодня, 3 дня, Неделя, Месяц.
· Постпрандиальный график (цветные точки по часам: 🔵 1ч, 🟣 2ч, 🟠 3ч).
· Статистика: средний сахар за выбранный период, общее количество ХЕ и другая аналитика.

4️⃣ Таймер и уведомления

· Таймер продолжает работать, даже если вы закрыли вкладку браузера (хранится в localStorage).
· Для получения звуковых/вибрационных уведомлений браузер может запросить разрешение — нажмите «Разрешить». Убедитесь, что звук на устройстве включен.

5️⃣ Свои продукты

· Нажмите «+» рядом со списком продуктов, заполните поля (название, углеводы на 100 г) и сохраните.
· Управлять списком (удалять) можно через кнопку «📝 Управление моими продуктами» на той же панели.

❓ Частые вопросы (FAQ)

Вопрос 1: Где взять мои коэффициенты (на ХЕ, чувствительности)?

Ответ: Только от вашего лечащего врача-эндокринолога. Это индивидуальные параметры, которые подбираются опытным путем. В приложении установлены безопасные значения по умолчанию (1.0 и 2.0), но их обязательно нужно заменить на свои.

Вопрос 2: Почему таймер не звонит?

Ответ: Проверьте, что вы нажали «▶️ Старт» после расчета. Также браузер мог заблокировать уведомления — нажмите на значок замочка/колокольчика в адресной строке и разрешите уведомления для этого сайта.

Вопрос 3: Как экспортировать данные (историю, графики) или перенести на другой телефон?

Ответ: Прямого экспорта пока нет. Все данные хранятся в вашем браузере (localStorage). При очистке кэша/истории браузера данные будут потеряны. Рекомендуем делать резервные копии скриншотами или вручную переписывать ключевые значения.

Вопрос 4: Работает ли приложение офлайн?

Ответ: Да, после первого открытия все необходимые файлы (HTML, CSS, JS) сохраняются в кэш браузера. Вы можете пользоваться калькулятором без доступа к интернету (кроме первоначальной загрузки страницы).

Вопрос 5: Приложение учитывает активный инсулин (IOB)?

Ответ: В текущей версии 6.0 — нет. Поле для ручного ввода активного инсулина появится в одном из следующих обновлений. Пока что старайтесь не делать частые коррекции (чаще 1 раза в 3-4 часа) без консультации с врачом.

⚠️ Важные предупреждения

⚠️ Предупреждение
1 Не заменяет врача. Приложение — это вспомогательный инструмент. Все решения по изменению доз инсулина, коэффициентов и целевых уровней принимаются только после консультации с эндокринологом.
2 Осторожно при высоком сахаре. При уровне сахара >15 ммоль/л перед введением инсулина необходимо проверить уровень кетонов (ацетона) в крови/моче. Высокая коррекция без кетонов опасна.
3 Гипогликемия не ждет. При сахаре <4.0 ммоль/л рекомендация приложения («съешьте 15 г углеводов») должна быть выполнена немедленно, а не после чтения дополнительных советов.
4 Хранение данных. Приложение не синхронизируется с облаком и не отправляет данные врачу. Вы полностью отвечаете за сохранность своей истории.
